# ChefHub

## Descripción General del Sistema
La aplicación ChefHub permite a los usuarios compartir recetas de cocina, comentarlas y puntuar las recetas de otros usuarios. Además, ofrece funcionalidades para buscar recetas y filtrarlas por categorías.

## Actores Principales
- **Usuario**: Persona que utiliza la aplicación para crear, comentar, puntuar y buscar recetas.
- **Administrador**: Encargado de moderar y gestionar los usuarios.
- **Moderador**: Encargado de moderar, gestionar las recetas y comentarios publicados.

## Objetivos del Sistema
- Permitir a los usuarios registrarse e iniciar sesión.
- Compartir recetas con otros usuarios.
- Dejar comentarios y puntuar recetas.
- Administrar la cuenta personal y las recetas publicadas.
- Realizar búsquedas avanzadas de recetas por nombre y categoría.
